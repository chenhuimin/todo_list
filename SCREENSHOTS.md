## 📱 界面元素详解

### 1. 侧边栏（Sidebar）

**位置：** 左侧固定宽度 256px

**组成：**
- Logo 区域（HealDocs + 图标）
- 导航菜单
  - Overview（单项）
  - Team Mates（可展开，包含 4 个成员）
  - Todo List（可展开，包含任务列表）

**交互：**
- 点击菜单项切换视图
- 点击成员名筛选该成员的任务
- 当前选中项高亮显示（蓝色背景）

**样式特点：**
- 白色背景
- 圆角图标按钮
- 悬停效果（hover:bg-gray-50）
- 展开/折叠动画

### 2. 顶部控制栏（Header）

**位置：** 顶部固定高度

**组成：**
- 左侧：标题 "Todo List" + 日期选择器
- 右侧：搜索框 + 通知图标 + 用户头像 + 添加按钮

**功能：**
- 日期选择器：切换查看不同日期的任务
- 搜索框：实时搜索任务标题
- 添加按钮：打开创建任务对话框

**样式特点：**
- 白色背景
- 底部边框
- 蓝色主按钮
- 图标按钮悬停效果

### 3. 任务看板（TaskBoard）

**位置：** 主内容区域

**组成：**
- 标签切换（Active Task / Completed）
- 任务卡片网格（响应式 1-3 列）

**布局：**
```css
/* 桌面 */
grid-cols-3    /* 3 列 */

/* 平板 */
grid-cols-2    /* 2 列 */

/* 手机 */
grid-cols-1    /* 1 列 */
```

**交互：**
- 点击标签切换视图
- 滚动查看更多任务
- 卡片悬停效果

### 4. 任务卡片（TaskCard）

**尺寸：** 自适应宽度，固定内边距

**组成：**
```
┌─────────────────────────┐
│ ☐ Task Title        ⋮  │  ← 头部（复选框 + 标题 + 菜单）
│                         │
│ Lorem ipsum dolor...    │  ← 描述（最多 2 行）
│                         │
│ 10:30 AM - 12:00 PM     │  ← 时间范围
│                         │
│ Assigned to: Name       │  ← 分配信息（可选）
└─────────────────────────┘
```

**配色：**
- 背景：淡色（100 色阶）
- 边框：深一级（200 色阶）
- 文字：深灰色

**交互：**
- 点击复选框切换完成状态
- 点击三点菜单显示操作选项（编辑/删除）
- 悬停显示阴影效果

### 5. 任务对话框（TaskDialog）

**位置：** 屏幕中央浮层

**尺寸：** 最大宽度 672px（max-w-2xl）

**组成：**

- 点击 Save 提交数据

## 🎬 用户操作流程

### 创建任务流程

1. 点击右上角 "Add New List" 按钮
2. 填写任务信息：
   - 输入标题（必填）
   - 输入描述（可选）
   - 选择颜色
   - 设置时间范围
   - 选择日期
   - 分配给团队成员（可选）
3. 点击 "Create Task" 按钮
4. 任务出现在 Active Task 列表中

### 编辑任务流程

1. 将鼠标悬停在任务卡片上
2. 点击右上角的三点菜单 ⋮
3. 选择 "Edit"
4. 修改任务信息
5. 点击 "Update Task" 按钮
6. 任务信息更新

### 标记完成流程

1. 点击任务卡片左上角的复选框 ☐
2. 复选框变为 ☑
3. 任务从 Active Task 移动到 Completed

### 删除任务流程

1. 点击任务卡片右上角的三点菜单 ⋮
2. 选择 "Delete"
3. 确认删除（浏览器弹窗）
4. 任务从列表中移除

### 搜索任务流程

1. 在顶部搜索框输入关键词
2. 实时过滤显示匹配的任务
3. 清空搜索框显示所有任务

### 按日期筛选流程

1. 点击日期选择器
2. 选择日期（Today/Yesterday/Tomorrow）
3. 显示该日期的任务

### 按成员筛选流程

1. 在侧边栏展开 "Team Mates"
2. 点击成员名称
3. 显示该成员的所有任务
